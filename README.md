# DIGITAL_BHEM-

Movie Ticket Booking System
A simple console-based Movie Ticket Booking System implemented in Java, provided by Digital Bhem. This project covers essential features such as user registration, movie listings, seat selection, payment processing, booking history, and an admin dashboard.

Features:
User Registration & Login: Secure account creation and login.
Movie Listings: View available movies, showtimes, and theaters.
Seat Selection: Select seats for your preferred showtime.
Payment Processing: Simulated payment gateway integration.
Booking History: Track and view past bookings.
Admin Dashboard: Manage movies, theaters, and bookings.
This project is a great starting point for learning Java and implementing core functionalities of a ticket booking system.
